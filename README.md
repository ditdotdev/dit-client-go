# Datadatdat Go Client

This project publishes a go client for [Datadatdat Server](https://github.com/datadatdat/datadatdat-server).
It is entirely autogenerated from the [OpenAPI specification](https://github.com/datadatdat/datadatdat-server/blob/master/openapi/datadatdat.yml).
Do not edit files directly, see the [Developing](DEVELOPING.md) section for more info.

## CI/CD Pipeline

This repository includes a comprehensive Pull Request 2 workflow with:
- Cross-platform testing (Ubuntu, Windows, macOS)
- Multi-version Go support (1.21, 1.22, 1.23)
- Security scanning and code quality checks
- Coverage reporting and performance benchmarks
